# PacMan-Java

## Installation

Clonez le projet :

git clone https://github.com/AyoubCharbaji/jeu_pacman.git   

cd pacman-java

## Compilation/Lancement

Compilez :

javac -cp . src/Resource/PacMan.java src/App.java

Lancez le jeu :

java -cp . App
